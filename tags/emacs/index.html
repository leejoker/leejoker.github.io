使用Spacemacs国内镜像

最近开始使用emacs写一些clojure的代码，但是不挂梯子的话，源的更新确实不快，因此决定到网上搜索一下怎么配置国内镜像，不过搜索结果大概都是下面这个样子：

# 基于emacs26
cd ~
git clone https://github.com/syl20bnr/spacemacs .emacs.d
修改
vim .emacs.d/core/templates/.spacemacs.template
