普通用户直接运行docker命令

2020年12月29日 18:13
，归类于 linux
目录
  • 具体操作
  • 补充说明

具体操作

  1. 添加docker group
$ sudo groupadd docker
  1. 添加用户到group
$ sudo gpasswd -a ${USER} docker
  1. 重启服务
$ sudo service docker restart

补充说明

这种方式适用于当root用户或者使用sudo可以执行，但普通用户无法执行的场景，不仅限于docker这一个服务，其他服务遇到此类情况的时候也不妨试一试~
