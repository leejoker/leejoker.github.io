Idea中的Git无法正常忽略

2020年1月28日 22:42
，归类于 技巧

这两天发现我在.gitignore文件中添加忽略后，每次提交已经被忽略的文件还是会出现在我的提交列表里，查了一下，应该是git缓存的问题，只要清理一下缓存就ok了。

$ cd $YOUR_PROJECT_DIR
$ git rm -r --cached .
$ git add .
$ git commit -m 'update .gitignore'
