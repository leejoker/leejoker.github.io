Ubuntu下添加ppa仓库

此处以安装atom为例，命令如下：

sudo add-apt-repository ppa:webupd8team/atom  
sudo apt-get update  
sudo apt-get install atom

这样就把atom的仓库添加并安装完成了。

然而，并没有这么简单

sudo: add-apt-repository : command not found
